El modelo SIR

Consideremos un modelo para describir la dinmica de un grupo de individuos de una poblacin con exposicin a una enfermedad que puede contagiarse entre los miembros de la poblacin. Esto puede modelarse como un sistema dinmico denominado \(SIR\) para una poblacin de \(N\) individuos en la que se considera la interaccin entre un conjunto de \(S\) individuos suceptibles de contraer la enfermedad, un conjunto \(I\) de individuos infectados y uno conjunto \(R\) de individuos recuperados de la enfermedad.

Este modelo tiene los siguientes supuestos:

  • la probabilidades de infectarse son iguales para todos los individuos de la poblacin;

  • la poblacin es homognea, es decir que los riesgos de infectarse son iguales para toos los suceptibles y que los tiempos para recuperarse son iguales para todos los infectados; y

  • el tamao \(N\) de la poblacin es constante.

El modelo maneja los diferentes conjuntos \(S\), \(I\) y \(R\) como si fueran compartimentos bien separados y considera que los individuos pueden pasr de uno a otro en el caso de que se enfermen (cambio \(S\rightarrow I\)) o que una vez enfermos se recuperen (cambio \(I\rightarrow\)). Ademas, se asume que un individuo no puede pasar del conjunto de suceptibles directamente al conjunto de recuperados.

Con estos supuestos y consideraciones, las ecuaciones diferenciales del modelo SIR son: \[ \begin{aligned} \frac{dS}{dt}&= -\beta \frac{I}{N} S\\ \frac{dI}{dt}&= \beta\frac{I}{N}S-\gamma I\\\ \frac{dR}{dt}&= \gamma I \end{aligned} \]

donde:

  • N=S+R+I

  • la cantidad \(\beta\frac{I}{N}\) representa la razn con que las personas salen del compartimento S (se infectan);

  • en la primera ecuacin \(dS\) representa el cambio debido a las personas que salen del compartimento \(S\) (el signo negativo se debe a que las personas salen)

  • en la segunda ecuacin \(dI\) representa el cambio debido a las personas que salen del compartimento \(I\) (una parte se debe a las personas que del compartimento \(S\) pasan al compartimento \(I\), y otra parte se debe a las personas que salen del compartimento \(I\) porque se recuperan);

  • la cantidad \(\gamma\) representa la razn con que las personas se recuperan.

library(deSolve)
library(reshape2)

initial_state_values <- c(S = 999999,  # Nmero de susceptibles inicial
                                       # 
                          I = 1,       # Se inicia con una persona infectada
                          R = 0)       # 


#razones en unidades de das^-1
parameters <- c(beta = 1,      # razn de infeccin
                gamma = 0.1)   # razn de recuperacin

#valores de tiempo para resolver la ecuacin, de 0 a 60 das
times <- seq(from = 0, to = 60, by = 1)   

sir_model <- function(time, state, parameters) {  
    with(as.list(c(state, parameters)), {# R obtendr los nombres de variables a
                                         # partir de inputs de estados y parametros
        N <- S+I+R 
        lambda <- beta * I/N
        dS <- -lambda * S               
        dI <- lambda * S - gamma * I   
        dR <- gamma * I                 
        return(list(c(dS, dI, dR))) 
    })
}

# poner la solucin del sistema de ecuaciones en forma de un dataframe
output <- as.data.frame(ode(y = initial_state_values, 
                            times = times, 
                            func = sir_model,
                            parms = parameters))

def deriv(y, t, N, beta, gamma):
    S, I, R = y
    dSdt = -beta * S * I / N
    dIdt = beta * S * I / N - gamma * I
    dRdt = gamma * I
    return dSdt, dIdt, dRdt

Comente acerca de los cambios que se observan en las curvas. Encuentre una relacin entre \(\beta\) y \(\gamma\) necesaria para que ocurra la epidemia. Para que haya una epidemia la fuerza de infeccin (\(\beta\)) debe ser suficientemente alta por un tiempo suficientemente largo (\(\gamma\) suficientemente bajo) de manera que se pueda transmitir el agente patgeno. A partir de este estudio se puede definir el coeficiente \(R_0\) de la infeccin.

A medida que se incrementa \(\beta\) el da de mximas infecciones ocurre antes, al igual que el estado de equilibrio. Esto ocurre ya que \(\beta\) est relacionado con la fuerza que tiene la enfermedad para infectar, de modo que a mayor valor de \(\beta\), mayor es su infecciosidad.

El valor \(R_0\) es conocido como el nmero reproductivo bsico, y determina la dinmica de una enfermedad regida por SIR. Se define como \(R_0 = \frac \beta \gamma\). Este indicador explica diferentes comportamientos segn su valor.

Si \(R_0 \in [0, 1)\) La enfermedad la enfermedad desaparecer con el tiempo, ya que cada caso existente causa, en promedio, menos de una nueva infeccin.

Si \(R_0 = 1\) la enfermedad se mantendr endmica (estable) en la poblacin, pero no causar una epidemia.

Si \(R_0 \in [0, \infty)\) la enfermedad se propagar y puede causar una epidemia, ya que cada caso existente causa, en promedio, ms de una nueva infeccin.
